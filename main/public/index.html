Como jogar Klotski

O objetivo é deslizar os blocos pelo tabuleiro até que a peça grande (vermelha) alcance a saída na parte inferior central.

Atalhos de Teclado

  • Espaço: Iniciar jogo
  • Esc: Parar jogo
  • Delete: Remover peça
  • Setas: Mudar Nível / Mover peça
  • F: Tela Cheia
  • S: Resolver puzzle
  • C: Limpar Tabuleiro
  • R: Resetar

Clique em uma peça para selecioná-la e use as setas ou o mouse para mover.
